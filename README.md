# Q Imageslider

[![Greenkeeper badge](https://badges.greenkeeper.io/nzzdev/Q-imageslider.svg?token=c13e57507403fee0c64955a752faf93fdcb6f5b553ec0165e3ed886447a411b9&ts=1551342785963)](https://greenkeeper.io/)

**Maintainer**: [manuelroth](https://github.com/manuelroth)

Q Imageslider is one tool of the Q toolbox to compare images with eachother.
Test it in the demo: https://editor.q.tools

## Installation

```bash
$ npm install
$ npm run build
```

## Development

Install the [Q cli](https://github.com/nzzdev/Q-cli) and start the Q dev server:

```
$ Q server
```

Run the Q tool:
```
$ node index.js
```

## Implementation details
The tool structure follows the general structure of each Q tool. Further information can be found in [Q server documentation - Developing tools](https://nzzdev.github.io/Q-server/developing-tools.html).
